# Test Plan - SQA Project

## 1. Introduction
This document outlines the test plan for the Automated Testing and CI project.

## 2. Objectives
- Validate login functionality
- Validate product browsing and cart functionality
- Measure test coverage and defect detection rate

## 3. Test Scope
### In Scope:
- Login page testing
- Product inventory page testing
- Cart functionality testing

### Out of Scope:
- Payment processing
- User registration
