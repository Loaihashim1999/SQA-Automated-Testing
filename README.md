# SQA Project: Automated Testing & CI

## Project Objectives
1. Design and develop an automated testing framework
2. Integrate testing with CI pipeline
3. Evaluate improvements in defect detection and test coverage

## Tools Used
- Python
- Selenium
- PyTest
- GitHub Actions
- SonarQube

## Setup
1. Create virtual environment: python -m venv venv
2. Activate: venv\Scripts\activate
3. Install dependencies: pip install -r requirements.txt
4. Run tests: pytest tests/ -v
